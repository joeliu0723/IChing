# V0.1 - 專案建立

日期：2026-07

## 完成項目

- 確立專案名稱：Project IChing（易經研究工作台）
- 確立 V1.0 開發目標
- 採用 Local First 架構
- 決定使用 JSON 作為唯一資料儲存格式
- 不使用 SQL Database
- 不使用 ORM
- 確立 Python 3.13 為開發語言
- 採用 PySide6 作為 GUI Framework
- 採用 Qt Designer 設計 UI
- 建立專案目錄規劃
- 建立文件架構

# Project IChing - Product Specification

Version: V1.0
Status: Frozen

---

# Purpose

本文件定義 Project IChing V1.0 的產品功能與操作流程。

本文件回答：

- 程式有哪些功能
- UI 應包含哪些畫面
- 使用者如何操作
- 每個功能應完成什麼工作

本文件不描述程式實作方式。

---

# Product Goal

Project IChing 是一套本機運行的易經研究工具。

主要用途：

- 起卦
- 解卦
- 保存研究紀錄
- 查詢歷史紀錄
- 驗證占卜結果
- 建立自己的易經研究資料庫

所有資料均保存於本機。

# Main Window

主視窗採用 QMainWindow。

畫面包含四個主要頁面（Tab）。

```
起卦
解卦
紀錄管理
設定
```

每個頁面彼此獨立。

---

# 起卦頁

功能：

建立新的占卜。

包含：

## 基本資料

- 姓名
- 性別
- 年齡
- 占卜問題

---

## 起卦方式

V1.0 支援四種方式：

1.

六爻輸入

2.

卦名輸入

3.

卦序輸入

4.

上下卦輸入

所有方式最後均產生相同結果。

---

## 六爻輸入

由下而上。

六列。

每列四種選項：

- 少陽
- 少陰
- 老陽
- 老陰

程式自動判斷：

- 本卦
- 變卦
- 動爻

---

## 顯示結果

完成排卦後顯示：

本卦：

- 卦序
- 卦名
- 上卦
- 下卦
- 六爻圖

變卦：

- 卦序
- 卦名
- 上卦
- 下卦
- 六爻圖

動爻：

列出所有動爻。

例如：

```
二、五
```

---

# 解卦頁

顯示：

本次占卜完整資訊。

包含：

基本資料

問題

本卦

變卦

動爻

---

## 解卦內容

提供大型文字編輯區。

使用者輸入：

自己的解卦內容。

---

## 驗證內容

提供第二個文字編輯區。

紀錄：

事後驗證。

---

## 驗證結果

固定四種：

- 未驗證
- 符合
- 部分符合
- 不符合

使用下拉選單。

---

## 收藏

Checkbox。

勾選後：

favorite = true

---

## 存檔

按下：

儲存。

建立 JSON。

若為既有紀錄：

更新 updated_at。

---

# 紀錄管理

列出所有占卜。

顯示：

- 日期
- 問題
- 本卦
- 變卦
- 收藏
- 驗證結果

---

## 搜尋

可搜尋：

- 問題
- 卦名
- 卦序
- 姓名
- 收藏
- 驗證結果

即時搜尋。

---

## 排序

支援：

- 日期
- 本卦
- 收藏
- 驗證結果

---

## 開啟

雙擊：

開啟紀錄。

---

## 編輯

可修改：

- 問題
- 解卦
- 驗證
- 驗證結果
- 收藏

不可修改：

- id
- created_at

---

## 刪除

永久刪除。

不提供垃圾桶。

刪除前需再次確認。

# Read Only Data

程式提供：

易經原文。

老師講義。

僅供閱讀。

# Error Handling

輸入錯誤：

顯示訊息。

不得造成程式崩潰。

JSON 損毀：

提示使用者。

略過損毀檔案。

程式繼續運作。
